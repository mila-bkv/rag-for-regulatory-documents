## Project Overview

This project implements an expert question-answering system over the Russian standard GOST R 59921 related to artificial intelligence systems.

The system combines:

- Dense retrieval using BGE-M3 embeddings
- Vector search using Qdrant
- Large language model Qwen3-1.7B for answer generation
- Inference performance evaluation (TTFT, ITL, TPS)

The goal is to study how different retrieval strategies affect answer quality and response latency.
